## 提交前重新格式化代码
如果在此项目中您希望提交之前重新格式化代码，则可以使用"提交工具窗口"中的选项进行此操作，该选项会启用 **⌘0**（macOS），或 **Ctrl+0**（Windows/Linux）：

![提交工具窗口中的选项](amend-cog.png)

单击此选项时，您将看到对话框：

![提交前格式化代码的复选框](reformat-code-checkbox.png)

选择  **格式化代码** 选项并按下 **OK**，在提交之前对文件进行重新格式化。 这将适用于您下一次提交任何编程语言的代码。
