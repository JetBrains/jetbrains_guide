  如果您想在 IntelliJ IDEA 最新版本的 Java 中使用预览功能，则需要将语言级别设置为"预览"。

  打开项目结构 **⌘;**（macOS）或 **Ctrl+Alt+Shift+S**（Windows/Linux），确保您选择了正确的 JDK，并从语言级别下拉中选择"预览"选项。
