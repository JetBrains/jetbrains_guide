JetBrains Rider 是一款基于 IntelliJ 平台和 ReSharper 的跨平台 .NET IDE。

Rider 让您可以高效地开发广泛的应用程序，包括：.NET 桌面应用程序、服务和库、Unity 和 Unreal Engine 游戏、Xamarin 应用、ASP.NET 和 ASP.NET Core web 应用程序。 On Windows, macOS, and Linux.
